# Variability ratio MCSE

Approximation of ratio variance based on independence of avg. se and
emp.se see wolter2007outstandR

## Usage

``` r
var.ratio.mcse(avg.se, emp.se, var.avg.se, var.emp.se)
```

## Arguments

- avg.se:

  Average SE

- emp.se:

  Emp SE

- var.avg.se:

  Variance of average SE

- var.emp.se:

  Variance of Emp SE
