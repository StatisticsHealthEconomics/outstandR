# Calculate trial variance continuous

Calculate trial variance continuous

## Usage

``` r
calculate_trial_variance_continuous(ald, tid, effect, verbatim = FALSE)
```

## Arguments

- ald:

  Aggregate level data. Data frame in long format.

- tid:

  Treatment ID

- effect:

  Effect name. String.

- verbatim:

  Print messages, logical

## Value

Numeric value of total variance.
