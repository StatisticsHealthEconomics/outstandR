# Calculate Trial Mean Continuous Data

Calculate Trial Mean Continuous Data

## Usage

``` r
calculate_trial_mean_continuous(ald, tid, effect, verbatim = FALSE)
```

## Arguments

- ald:

  Aggregate level data. Data frame in long format.

- tid:

  Treatment ID

- effect:

  Effect name. String.

- verbatim:

  Print messages, logical

## Value

Numeric mean value.
