# Get reference treatment

Get reference treatment

## Usage

``` r
get_ref_trt(ref_trt, trt, ipd_trial, ald_trial)
```

## Arguments

- ref_trt:

  Reference treatment

- trt:

  Treatment

- ipd_trial:

  A dataframe of IPD

- ald_trial:

  A dataframe of ALD

## Value

String of reference treatment name.
