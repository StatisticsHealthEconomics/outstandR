# Calculate Trial Mean Count Data

Calculate Trial Mean Count Data

## Usage

``` r
calculate_trial_mean_count(ald, tid, effect, verbatim = FALSE)
```

## Arguments

- ald:

  Aggregate level data. Data frame in long format.

- tid:

  Treatment ID

- effect:

  Effect name. String.

- verbatim:

  Print messages, logical

## Value

Numeric mean value.
