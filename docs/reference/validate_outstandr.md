# Input data validator

<div class="ref-description section level2">

Input data validator

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
validate_outstandr(ipd_trial, ald_trial, strategy, CI, scale)
```

</div>

</div>

<div class="section level2">

## Arguments

-   ipd_trial:

    Individual patient data

-   ald_trial:

    Aggregate level data

-   strategy:

    An object of class `strategy` created by functions such as
    `strategy_maic()`, `strategy_stc()`, or `strategy_mim()`. Contains
    modelling details like the formula and family.

-   CI:

    Confidence interval

-   scale:

    Outcome scale

</div>

<div class="section level2">

## Value

No return value, called for side effects

</div>
