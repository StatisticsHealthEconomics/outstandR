# Data set for individual level patient data

This data set contains simulated patient covariate and outcome values.

## Format

A data list including the variables needed for a outstandR basic
example. The variables are as follows:

- X1:

  Numeric covariate

- X2:

  Numeric covariate

- X3:

  Numeric covariate

- X4:

  Numeric covariate

- trt:

  Integer treatment identifier

- y:

  Integer binary outcome

## Source

Simulated data
